redis

  • Redis = REmote DIctionary Server

  • In-memory data store, often used as a cache

  • Keeps all data RAM → very fast

  • Stores data as key-value pairs

  • Values can be strings / lists / sets / hash etc

  • Periodically save snapshots or append logs to disk to recover after restarts

  • Use cases:

    • Caching — store the results of expensive database queries or API calls to speed up response time.

    • Session storage — websites store user sessions in Redis so they can be shared across multiple web servers.

  • Wordpress normally stores cached data in the mariadb database or PHP memory, which can be slow

    • Redis Object Cache plugin replaces this with Redis, so WordPress can:

      • Store query results, options, and object data in RAM

      • Retrieve them almost instantly

      • Reduce database load
