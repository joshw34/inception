SQL Commands

SHOW DATABASES

  • List all databases

 

USE wordpress

  • Select database

 

SHOW TABLES

  • List tables

 

SELECT * FROM table

  • Get all columns (* = wildcard)

 

SELECT COUNT(*) FROM table

  • Count rows

 

DESCRIBE table

  • Show table structure

 

CREATE USER 'user'@'host' IDENTIFIED BY 'pass'

  • @ separates username from hostname

  • 'localhost' = local only

  • '%' = any host (wildcard)

  • IDENTIFIED BY = set password

 

GRANT ALL PRIVILEGES ON db.* TO 'user'@'host'

  • db.* = all tables in database (* = wildcard)

  • *.* = all databases and tables

 

FLUSH PRIVILEGES

  • Reload permissions

 

SELECT User, Host FROM mysql.user

  • mysql.user = system table (database.table)

  • WHERE column = 'value'

  • Filter results

 

LIMIT 10

  • Return only 10 rows

 

LIKE 'pattern%'

  • Pattern matching (% = wildcard)

 

Key Symbols

  • @ = user@host separator

  • % = wildcard (any/all)

  • * = all (columns/tables/rows)

  • . = database.table separator

  • = = equals comparison

  • ; = statement terminator

  • ' ' = string values
